# 闲鱼代发店铺运营方案
> 老板：豆云昊 | 运营：AI 数字员工 | 模式：纯代发（无货无仓）

---

## 1. 🎯 经营定位

### 模式
```
闲鱼发布商品 → 买家下单付款 → 我转发订单给供应商 → 供应商发货
                                               ↑
                                          （代发/一件代发）
```

### 优势
- 零库存风险，不压资金
- 可同时上架多品类测试市场
- 闲鱼流量成本低，免费流量充沛

### 核心指标
| 指标 | 目标 | 说明 |
|------|:----:|------|
| 毛利率 | ≥ 20% | 售价 − 进货成本 − 平台费用 |
| 回复率 | 100% | 闲鱼权重关键指标 |
| 回复时长 | < 5 分钟 | 影响曝光排名 |
| 发货时效 | 24h 内 | 供应商发货后回传单号 |

---

## 2. 📦 选品策略

### 选品原则（代发适用）
- ✅ 轻小件（运费低、退货少）
- ✅ 标品（无需解释、售后少）
- ✅ 高溢价空间（进货价 < 售价 60%）
- ✅ 有现货、不断货的供应商
- ❌ 大件/重货（运费吃利润）
- ❌ 易损耗品（售后率高）

### 推荐赛道
| 品类 | 理由 | 1688 均价 |
|------|------|:---------:|
| 家居收纳 | 刚需、轻小、复购高 | 5-30 元 |
| 数码配件 | 手机壳/充电器/数据线 | 3-20 元 |
| 办公文具 | 高溢价、不退换 | 2-15 元 |
| 个护小物 | 高频消耗 | 5-25 元 |
| 宠物用品 | 增速快、粘性高 | 8-40 元 |

## 3. 🔗 供应商对接

### 代发渠道
| 平台 | 特点 | 对接方式 |
|------|------|----------|
| **1688 一件代发** | 最成熟，支持自动 | 直接下单留买家地址 |
| **拼多多** | 价格低，注意隐蔽 | 转单时隐藏代发信息 |
| **淘宝/天猫** | 适合补货 | 同上 |
| **采源宝** | 1688 官方代发工具 | 手机端操作 |

### 合作流程
```
我在 1688 选好品 → 联系供应商确认:
  ✓ 是否支持一件代发
  ✓ 发货时效（最好 24h 内）
  ✓ 是否支持无痕发货（不夹带店铺信息）
  ✓ 退换货政策
  ✓ 有无拼多多/淘宝比价限制
→ 确认后录入商品库
```
